Release Notes

Release Date: 29 March 2009
This is the 0.9.6 development release

Project: SwingX

This release is focused on code and API cleanup. Just few bigger changes. Mostly API cleanup and deprecated code removal.

Paiters

Reworked generics to work correctly in all usage scenarios.

RepaintManagerX

Updated manager to allow use of other repaint managers.

JXMonthView

Dropped pre-rendering paining code. Fixed issues with padding and other UI artefacts.

JXStatusBar

Fixed issues with padding and other UI artefacts.

JXLoginPane

Password store now provides passwords without need to start typing first.
