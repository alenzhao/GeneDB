Jogra

Jogra is an overarching application which can be used to call other applications such as the term rationaliser. The Term Rationaliser allows the user to correct and 'curate' terms names pertaining to a selected organism (e.g., product names, controlled curation terms). For instance, slightly different spellings for the same term can all be rationalised into the correct name.
