Jogra

Jogra is a free application that contains several tools related to data curation and correction in databases that implement the Chado schema. It is maintained by the Pathogen Informatics Group at the Wellcome Trust Sanger Institute, UK. You can connect to a suitable chado-based database by specifying the connection details at the start.

Launch Jogra

Organism Chooser

The Organism Chooser application allows the user to select the organism(s) that he or she wants to work with. This sets the scope for all the tools contained within Jogra.

Term Rationaliser

The Term Rationaliser is a controlled vocabulary curation tool. It allows the user to select a vocabulary (e.g., Gene Products) and correct any redundant or erroneous names. For instance, slightly different spellings for the same term can all be rationalised into the correct name easily, without having to go through the database. Additionally, you can restrict the scope of your changes by selecting the organisms you want to make the changes in.

More information on how to launch and use the Term Rationaliser can be found here.
